This website uses Vue components and props which, simply put, makes elements reusable and allows for data within the component to be changed easily. For example, we have several cats that are listed by ids and are queried from the cats api so our changing data would be the cats' breed ids, this way, we can see different cats along with their images, descriptions, and other unique information. In our Vue file, we have the loadNextImage method that takes in the breed id and outputs an array with all of their images and data. The images are flipped through by a method called slideshow which goes through the gallery of cats images. All this information is inputed into the template which is a special feature of the Vue component. There, we can create an html div that can be used in the actual html page. In this case, our component name is 'catshow' and our prop is 'whichcat'. We use this in the html page by doing < catshow whichcat="somecatidhere">< /catshow> which then outputs the specific cat and it's data. For this project, I also added a star rating to visualize information like how child friendly the cat is and their energy levels using a star rating system. For this, I created a loadStars() method that takes in a value and contains a switch case that returns the correct html snippet and gets added in the template which then shows up on the main html page. When you click the navigation bar items above, you can flip through the different cats and only the cat breed that is selected appears.
